# 🎂 Happy Birthday Cheenuuuu ❤️

A beautiful interactive birthday website built using React + Vite.

## ✨ Features

- Fake Incoming Voice Call
- Password Lock Screen
- Animated Loading Screen
- Hero Section
- Birthday Story Book
- Memory Gallery
- FaceTime Call
- Gift Box Animation
- Birthday Letter
- Music Player
- Cake Celebration
- Fireworks
- Beautiful Ending Screen

---

## 🚀 Installation

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build project

```bash
npm run build
```
